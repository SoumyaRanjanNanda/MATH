## 1. **Definition**

A **function** is a relation between two sets where **each input has exactly one output**.

ðŸ‘‰ Think of a machine: you give it an input, it processes, and gives exactly one output.

Example:

$$
f(x) = x^2
$$

If you put $2$, you get $4$. If you put $-2$, you also get $4$.
But for one input, youâ€™ll never get two different outputs.

---

## 2. **Representation**

Functions can be written in many ways:

* **Equation form**: $f(x) = 2x + 3$
* **Set form**: $f: X \to Y$
* **Graph form**: curve or line on coordinate plane

---

## 3. **Important Terms**

* **Domain**: All possible inputs (x-values).
* **Co-domain**: Set of possible outputs (Y-values).
* **Range**: Actual outputs we get after applying the function.

Example:

$$
f(x) = x^2
$$

* Domain = All real numbers ($\mathbb{R}$)
* Co-domain = $\mathbb{R}$
* Range = $[0, \infty)$

---

## 4. **Types of Functions**

1. **One-to-One (Injective)** â†’ Each input has a unique output, and no two inputs give the same output.
   Example: $f(x) = 2x + 3$

2. **Onto (Surjective)** â†’ Every element of the co-domain has some input mapping to it.
   Example: $f(x) = x^3$ (covers all $\mathbb{R}$)

3. **Bijective** â†’ Both One-to-One and Onto (invertible function).
   Example: $f(x) = x + 5$

4. **Even Function** â†’ $f(-x) = f(x)$.
   Example: $f(x) = x^2$

5. **Odd Function** â†’ $f(-x) = -f(x)$.
   Example: $f(x) = x^3$

6. **Polynomial Function** â†’ $f(x) = ax^n + bx^{n-1} + \dots + c$

7. **Exponential Function** â†’ $f(x) = a^x$

8. **Logarithmic Function** â†’ $f(x) = \log_a(x)$

9. **Trigonometric Functions** â†’ $\sin x, \cos x, \tan x, \dots$

---

## 5. **Operations on Functions**

* **Addition**: $(f+g)(x) = f(x) + g(x)$
* **Multiplication**: $(f \cdot g)(x) = f(x) \cdot g(x)$
* **Composition**: $(f \circ g)(x) = f(g(x))$

Example:

$$
f(x) = x+1, \quad g(x) = 2x
$$

$$
(f \circ g)(x) = f(g(x)) = f(2x) = 2x + 1
$$

---

## 6. **Inverse of a Function**

If $f(x)$ is bijective, it has an **inverse function** $f^{-1}(x)$.
It reverses the mapping:

$$
f(f^{-1}(x)) = x, \quad f^{-1}(f(x)) = x
$$

Example:

$$
f(x) = 2x+3
$$

Inverse:

$$
f^{-1}(x) = \frac{x-3}{2}
$$
